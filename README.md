# LoftCoin

This is a small education project called LoftCoin

Stack of technology:
* Kotlin
* OkHttp
* Retrofit
* Moshi
* Picasso
* ViewModel
* LiveData
* Dagger 2
* FragmentFactory
* Navigation Component
* Room
